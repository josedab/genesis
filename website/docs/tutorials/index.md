## What You'll Learn

Each tutorial teaches specific skills:

| Tutorial | Skills |
|----------|--------|
| Customer Data | Basic synthesis, quality evaluation, constraints |
| Healthcare Compliance | Privacy settings, HIPAA compliance, audit reports |
| ML Pipeline | Data augmentation, model training, Pipeline API |
| Testing | Test fixtures, CI/CD integration, edge cases |

## Prerequisites

Before starting tutorials, you should:

- Have Python 3.8+ installed
- Have basic Python/pandas knowledge
- Have Genesis installed: `pip install genesis-synth[pytorch]`

## Tutorial Format

Each tutorial includes:

1. **Goal** - What you'll build
2. **Prerequisites** - What you need
3. **Step-by-step instructions** - With runnable code
4. **Expected output** - So you know you're on track
5. **Next steps** - Where to go from here

All code is copy-paste runnable. No placeholders.
